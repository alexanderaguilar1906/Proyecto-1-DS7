## 📁 Estructura del Proyecto

```
ecommerce/
├── index.html              ← Página principal (Inicio)
├── css/
│   └── style.css           ← Estilos HTML5 & CSS3 completos
├── js/
│   ├── products.js         ← Base de datos de productos
│   ├── cart.js             ← Lógica del carrito
│   └── main.js             ← Funciones generales
└── pages/
    ├── tienda.html         ← Tienda con filtros y productos
    ├── nosotros.html       ← Historia, valores y equipo
    └── contacto.html       ← Formulario + Mapa + FAQ
```

---

## 🚀 Cómo ejecutar

1. Abrir la carpeta `ecommerce/`
2. Hacer doble clic en `index.html`
3. ¡Listo! No requiere servidor ni instalación.

> Para mejor experiencia, se recomienda usar **Live Server** en VS Code.

## 🎨 Tecnologías usadas

- **HTML5**: Semántica, accesibilidad, audio, video, iframe
- **CSS3**: Custom Properties, Grid, Flexbox, Keyframes, Media Queries, Backdrop Filter
- **JavaScript**: Carrito funcional con localStorage, filtros, formulario, animaciones con IntersectionObserver
- **Fuentes**: Google Fonts (Cormorant Garamond + DM Sans)
